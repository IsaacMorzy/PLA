# X Thread #1 — 4 Trust Checks Before You Donate

## Objective
Educate donors with a repeatable pre-donation evaluation framework.

## Thread Copy
**1/** Before you donate to any campaign, run 4 quick trust checks. It takes under 2 minutes and improves donation confidence.

**2/** Check #1: Is the need specific?
A credible campaign explains who needs help, what is needed, and why now.

**3/** Check #2: Is progress clear?
You should be able to see raised amount, goal, and current momentum without guessing.

**4/** Check #3: Is there visible activity?
Look for signs the campaign is active and being supported in real time.

**5/** Check #4: Is the next action obvious?
If the flow from review → donate is confusing, trust drops.

**6/** We built PeaceLeague Africa around these checks so donors can decide faster and with more confidence.

**7/** Browse live campaigns: https://peaceleague.africa/campaigns
