# LinkedIn Post #2 — Creator Credibility (Consideration → Decision)

## Objective
Increase campaign creation intent by showing creators what improves donor trust immediately.

## Final Copy
Most campaigns don’t fail because the cause is weak.
They fail because the story is unclear.

If you’re launching a fundraiser, these 3 elements usually decide whether donors trust it:
1) A specific problem statement (who needs help + why now)
2) A clear funding target with visible progress
3) A clean donation path with no ambiguity

When donors understand quickly, support starts faster.

If you’re preparing a campaign, use this framework before publishing.
